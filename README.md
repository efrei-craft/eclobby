# EC Lobby

Plugin that manages all functions in Efrei Craft's Hub.

## Permissions

| Permission      | Description                                |
|-----------------|--------------------------------------------|
| `eclobby.lobby` | Allows the player to use `/lobby` command. |
| `eclobby.join`  | Allows the player to use `/join` command.  |
